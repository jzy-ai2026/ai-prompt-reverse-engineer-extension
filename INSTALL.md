# AI Prompt Reverse Engineer 安装与使用教程

这个插件用于在网页图片或本地图片上反推 AI 生成提示词，并输出中文 Prompt 和结构化 JSON。它也内置 Nano Banana Pro 提示词助手，可以把中文想法、参考图、画幅、分辨率和限制条件整理成可直接复制的英文提示词。

适用浏览器：

- Chrome 120+
- Edge 120+

## 一、下载插件

打开 GitHub 仓库：

```text
https://github.com/jzy-ai2026/ai-prompt-reverse-engineer-extension
```

下载插件压缩包：

```text
release/ai-prompt-reverse-engineer-extension-latest.zip
```

下载后先解压。解压完成后，文件夹里应该能直接看到这些文件：

```text
manifest.json
background.js
content.js
sidepanel.html
assets/
icons/
```

注意：加载插件时要选择“包含 manifest.json 的那个文件夹”，不要选择 zip 文件本身，也不要选择它的上一级目录。
如果你从 GitHub 点的是绿色 `Code -> Download ZIP`，那下载的是源码包，不能直接当插件安装；请使用 `release/` 目录里的插件 zip。

## 二、在 Chrome 中安装

1. 打开 Chrome。
2. 在地址栏输入：

```text
chrome://extensions
```

3. 打开右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择刚才解压出来的插件文件夹。
6. 页面里出现 `AI Prompt Reverse Engineer`，说明安装成功。

## 三、在 Edge 中安装

1. 打开 Edge。
2. 在地址栏输入：

```text
edge://extensions
```

3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择刚才解压出来的插件文件夹。
6. 页面里出现 `AI Prompt Reverse Engineer`，说明安装成功。

## 四、首次配置 API

安装成功后，点击浏览器工具栏里的插件图标，打开插件侧边栏。

进入“设置”页面，填写：

```text
API Base URL: https://ai.leihuo.netease.com/v1
Model: gemini-3.1-pro-preview-customtools
API Key: 你的公司 AI 网关 Bearer Token
```

填写完成后点击保存。

说明：

- API Key 只保存在你自己的浏览器本地。
- API Key 不会上传到 GitHub，也不会写入插件代码。
- 如果不知道 API Key，请联系公司内部 AI 网关管理员。

## 五、选择反推模板

侧边栏工作区顶部有“模板切换”下拉框。右键反推、上传图片反推和批量逐张反推会使用当前选中的单图模板；“同风格”多图分析固定使用内置“多图风格反推”模板。

内置模板包括：

- JSON提示词反推
- 风格提取
- 多图风格反推
- 人物参考表
- 角色阵容设计
- 物体道具参考表
- 资产拆分
- 高品质 3D 渲染
- 电影大师
- 高清细化
- 线稿
- 色稿

进入“模板”页面可以：

- 查看内置模板的 System Prompt
- 复制内置模板为自定义模板
- 新增、编辑、删除自定义模板
- 将任意模板设为当前模板

说明：内置模板不可直接编辑或删除，避免误删；需要改动时请先复制为自定义模板。

## 六、反推图片提示词

1. 打开一个包含图片的网页。
2. 在目标图片上点击右键。
3. 点击菜单里的“反推提示词”。
4. 首次使用时，插件会提示：

```text
图片/URL 将发送至你配置的 API 服务
```

5. 点击“本次允许”或“记住选择”。
6. 等待模型分析完成。
7. 侧边栏会显示：

- 参考图片
- 中文 Prompt 预览
- PromptDocument JSON
- 历史记录

## 七、使用 Nano Banana Pro 提示词助手

提示词助手适合“我已经有创意或参考图，现在想得到一条高质量英文出图 Prompt”的场景。

### 1. 进入助手

1. 点击浏览器工具栏里的插件图标，打开侧边栏。
2. 点击顶部第二个入口“提示词助手”。
3. 看到“提示词助手”页面后，开始填写需求。

### 2. 选择任务类型

| 模式 | 用途 |
| --- | --- |
| 自动判断 | 不确定该选哪种时使用 |
| 文生图 | 只根据文字想法生成英文 Prompt |
| 图文生成 | 结合参考图和文字生成新图 Prompt |
| 编辑图片 | 面向改图任务，强调保留或替换哪些视觉元素 |

### 3. 填写中文想法

在“想法”里写清楚主体、场景、风格、构图和需要出现的文字。例如：

```text
生成一个国风武侠场景，角色站在雨夜古街中央，画面有电影海报质感，人物穿黑色斗篷，远处有灯笼和湿润石板路。
```

### 4. 设置画幅和分辨率

常用设置：

- 横版封面：`16:9 + 2K`
- 竖版海报：`4:5 + 2K`
- 手机竖屏：`9:16 + 2K`
- 方图头像：`1:1 + 2K`
- 最终精修：对应画幅 + `4K`

### 5. 添加参考图

在“参考图”区域拖入、粘贴或点击上传图片，最多 6 张。缩略图支持逐张复制、删除和拖拽排序，每张图都可以指定参考角色：

- 身份参考：保留人物、角色或产品身份。
- 风格参考：参考画风、质感、调色和视觉语言。
- 构图参考：参考镜头角度、版式和空间关系。
- 场景参考：参考地点、环境和氛围。
- 产品参考：参考产品造型、结构、品牌文字和识别特征。
- 文字参考：参考画面文字、标题样式和字体风格。
- 材质参考：参考材质、表面处理和细节密度。

如果要保留人物、角色或产品一致性，打开“身份锁定”。它会要求模型不要改年龄、脸型、五官比例、产品结构和核心识别特征。

### 6. 填写规格补充

“规格补充”适合写限制条件，例如：

```text
画面文字必须是“风起长安”。
不要出现现代建筑。
保持角色脸型、年龄和眼距不变。
品牌 Logo 必须清晰，不要改字。
```

### 7. 生成并检查

点击“生成提示词”后，助手会输出：

- 最终英文提示词：直接复制到 Nano Banana Pro。
- 中文核对：检查英文 Prompt 是否准确表达中文需求。
- 核对清单：确认主体、参考图角色、场景、风格、镜头、身份锁定、负面约束、画幅和分辨率。
- 可能需要确认：列出还不够明确的点。
- 负面约束：提醒哪些内容不要生成或不要改变。

确认无误后，点击复制按钮，粘贴到 Nano Banana Pro 使用。

## 八、复制结果

分析完成后可以复制两种内容：

- 复制 Prompt：适合直接粘贴到 AI 绘画或视频生成工具中。
- 复制 JSON：适合保存结构化分析结果，或继续做二次编辑。

## 九、多图参考

1. 打开包含参考图的网页。
2. 在第一张参考图上点击右键。
3. 点击菜单里的“添加到多图参考”。
4. 对其他参考图重复上面的操作。
5. 也可以打开插件侧边栏，使用“多选”一次选择多张本地图片，或使用“文件夹”导入一个文件夹里的图片。
6. 打开插件侧边栏，在“多图参考”里确认图片。
7. 队列中至少有 2 张图片后，可以选择：

- 同风格：固定使用“多图风格反推”模板，提取多张图共享的视觉风格，结果标记为 `source.type=style_common`。
- 批量：逐张按当前单图模板分析，每张图生成一条历史记录，工作区显示最后一张结果。

说明：

- 多图参考最多保留 6 张图。
- 可以点击单张缩略图右上角按钮移除。
- 可以复制单张缩略图，方便把同一张图同时作为身份参考和风格参考。
- 可以拖拽缩略图调整顺序，`@图片1`、`@图片2` 会按新顺序重新编号。
- 可以点击垃圾桶按钮清空队列。

## 十、修改提示词

底部输入框支持两种修改方式。

### 方式一：快捷字段修改

适合快速改某个字段：

```text
style=赛博朋克
色调=暖色
主体=一只橘猫
光影=电影感逆光
```

这种方式会在本地立即修改 JSON，不会额外调用 API。

### 方式二：自然语言修改

适合一次性改多个维度：

```text
把画面风格改成赛博朋克，色调换成暖色，主体换成一只猫
```

这种方式会调用 API，让模型返回修改后的完整 JSON。

### 编辑模式

输入框上方会显示当前模式：

- 文本编辑：只把当前 JSON 和修改指令发给模型，更快更省 token。
- 视觉参考：会把原图上下文一起发给模型，适合要求保持原图风格或根据图片细节修改。

默认使用“自动判断”。当指令包含以下关键词时，会自动升级为视觉参考：

```text
参考原图
按原图
保持原图风格
根据图片
像原图一样
```

也可以手动切换为“固定文本编辑”或“固定视觉参考”。

## 十一、历史记录

插件会保留最近 20 条结果。

进入“历史记录”页面可以：

- 查看之前反推过的 Prompt
- 点击历史项恢复到工作区，包括 Prompt 和参考图
- 删除单条历史
- 清空全部历史

## 十二、常见问题

### 1. 右键菜单没有出现

请刷新网页后再试一次。插件菜单只会在图片上右键时出现。

### 2. 提示 API 请求失败

请先检查设置页：

```text
API Base URL 是否为：https://ai.leihuo.netease.com/v1
Model 是否为：gemini-3.1-pro-preview-customtools
API Key 是否填写正确
```

如果仍然失败，截图红色错误框发给插件维护者排查。

### 3. 提示图片跨域限制

部分网站的图片无法被插件直接读取。可以这样处理：

1. 先把图片下载到本地。
2. 打开插件侧边栏。
3. 在“参考图片”区域点击上传按钮。
4. 选择本地图片重新分析。

### 4. 反推出的内容不是中文

请确认已经安装最新版本插件。新版会要求模型输出简体中文 Prompt。

### 5. 更新插件版本

1. 下载新版 zip。
2. 解压并覆盖旧文件夹，或解压到一个新文件夹。
3. 打开：

```text
chrome://extensions
```

4. 找到 `AI Prompt Reverse Engineer`。
5. 点击卡片上的“重新加载”按钮。

如果你换了新的文件夹，也可以先移除旧插件，再重新“加载已解压的扩展程序”。

## 十三、卸载插件

进入：

```text
chrome://extensions
```

找到 `AI Prompt Reverse Engineer`，点击“移除”即可。
