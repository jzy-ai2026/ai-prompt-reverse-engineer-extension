# AI Prompt Reverse Engineer 功能介绍与使用教程

## 一句话介绍

AI Prompt Reverse Engineer 是一款 Chrome/Edge 侧边栏插件，用来从网页图片、本地图片或多张参考图中反推出可复用的 AI 图像生成提示词，并输出中文 Prompt、结构化 JSON 和 Nano Banana Pro 可直接使用的英文提示词。

## 产品定位

这不是一个简单的图片描述工具，而是面向 AI 绘画、视觉参考分析、概念设计和素材制作的 Prompt 工作台。用户在浏览网页寻找参考图时，可以直接右键分析图片，也可以在侧边栏中上传、粘贴或拖拽图片，把视觉信息整理成可控、可修改、可再次生成的提示词文档。

## 插件界面

插件顶部按常用工作流分成 5 个入口：

| 入口 | 用途 |
| --- | --- |
| 图片反推工作区 | 右键网页图片、上传本地图片、批量分析多张图，生成中文 Prompt 和 JSON |
| Nano Banana Pro 提示词助手 | 把中文想法、参考图和限制条件整理成英文提示词，并提供中文核对 |
| 历史记录 | 查看、恢复或删除之前生成过的结果 |
| 模板 | 查看内置模板，复制并改成自定义模板 |
| 设置 | 配置 API Base URL、模型名和 API Key |

## 核心功能

### 1. 网页图片右键反推

在任意包含图片的网页中，右键目标图片即可选择“反推提示词”。插件会捕获图片地址、来源页面和页面标题，并在侧边栏中完成分析，适合边浏览素材边整理 Prompt。

### 2. 本地图片导入分析

侧边栏支持上传图片、粘贴截图和拖拽图片。导入后会进入分析流程，适合处理本地参考图、截图、设计稿和临时素材。

### 3. 多图参考与批量分析

用户可以把 2 到 6 张参考图加入多图队列，用于提取共同风格、批量逐张反推，或作为提示词助手的参考图。

- **逐张操作**：每张缩略图都可以单独删除、复制和拖拽排序，方便精细增减参考图。
- **同风格**：固定使用“多图风格反推”模板，提取多张图共享的视觉风格。
- **批量**：按当前单图模板逐张反推，并为每张图片生成历史记录。
- **助手参考图**：在提示词助手里给每张图指定“身份参考”“风格参考”“构图参考”等角色。

### 4. 专业模板库

插件内置 12 套专业模板，覆盖常见视觉工作流：

- JSON 提示词反推
- 风格提取
- 多图风格反推
- 人物参考表
- 角色阵容设计
- 物体道具参考表
- 资产拆分
- 高品质 3D 渲染
- 电影大师
- 高清细化
- 线稿
- 色稿

用户也可以复制内置模板后修改，或新增自己的模板，用不同 System Prompt 固化团队常用输出格式。

### 5. 中文 Prompt 与 JSON 双视图

分析完成后，插件会同时提供自然语言 Prompt 预览和结构化 JSON 编辑视图。Prompt 适合快速复制到图像或视频生成模型，JSON 适合做字段级修改、二次加工和长期归档。

### 6. 自然语言二次修改

底部输入框支持直接用中文修改结果，例如：

```text
把色调改成暖色，主体换成一只猫
```

也支持快捷字段修改：

```text
style=赛博朋克
```

当指令包含“参考原图”“按原图”“保持原图风格”或 `@图片1` 这类引用时，插件会自动切换到视觉参考模式，让模型结合原图继续改写 PromptDocument。

### 7. Nano Banana Pro 提示词助手

提示词助手面向“我已经知道想要什么，但需要一条高质量英文出图 Prompt”的场景。它会把中文想法、参考图角色、画幅比例、分辨率和限制条件整理成 Nano Banana Pro 可直接使用的英文提示词。

助手输出包含：

- **最终英文提示词**：可直接复制到 Nano Banana Pro。
- **中文核对**：把英文 Prompt 的含义回译成中文，方便检查是否跑偏。
- **核对清单**：检查主体、参考图角色、场景、风格、镜头、身份锁定、负面约束、画幅和分辨率。
- **可能需要确认**：列出模型认为仍然含糊的点。
- **负面约束**：提示哪些内容不要改、不要生成或不要混合。

### 8. 助手历史

提示词助手会保存最近 20 条结果。历史项可以恢复到助手界面继续查看、复制或再次修改，也可以单条删除或全部清空。

### 9. OpenAI 兼容网关配置

插件支持配置 OpenAI 兼容 API Base URL、模型名和 API Key，可接入公司内部 AI 网关或其他兼容服务。默认模型配置面向视觉理解场景，可在设置中切换或扩展模型预设。

### 10. 本地隐私控制

API Key 只保存在浏览器本地 `chrome.storage.local`。首次分析图片时，插件会请求用户确认是否允许将图片或图片 URL 发送到已配置的 API 服务，避免在不知情的情况下上传参考图。

## 提示词助手教程

### 适合什么时候用

当你不是在“反推某张图片”，而是在准备写一条更完整的出图提示词时，优先使用提示词助手。

常见场景：

- 纯文字生成一张海报、角色图、产品图或场景图。
- 给一张或多张参考图指定不同用途，例如人物身份参考、风格参考、构图参考。
- 把中文需求整理成专业英文 Prompt。
- 需要确认英文 Prompt 有没有误解中文需求。
- 需要让模型严格保留人物、角色或产品身份。

### 步骤一：进入提示词助手

1. 点击插件图标打开侧边栏或网页内浮动面板。
2. 点击顶部第二个入口“Nano Banana Pro 提示词助手”。
3. 看到标题“提示词助手”后，就可以开始填写需求。

### 步骤二：选择任务类型

| 模式 | 什么时候用 |
| --- | --- |
| 自动判断 | 不确定该选哪种时使用，插件会根据是否有参考图和文字需求自动组织任务 |
| 文生图 | 没有参考图，只根据文字想法生成英文 Prompt |
| 图文生成 | 有参考图，希望结合身份、风格、构图、材质或产品信息生成新图 |
| 编辑图片 | 面向改图任务，强调保留或替换哪些视觉元素 |

### 步骤三：填写“想法”

在“想法”里写清楚你要生成什么、主体是谁、画面在哪里、风格是什么、是否需要文字。

示例：

```text
生成一个国风武侠场景，角色站在雨夜古街中央，画面有电影海报质感，人物穿黑色斗篷，远处有灯笼和湿润石板路。
```

### 步骤四：设置画幅和分辨率

画幅比例支持：

```text
1:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9
```

分辨率支持：

- `1K 草图`：适合快速测试。
- `2K 推荐`：适合默认出图，质量和速度比较均衡。
- `4K 高细节`：适合最终海报、产品图和细节要求高的画面。

常用组合：

| 用途 | 推荐设置 |
| --- | --- |
| 横版封面 | `16:9 + 2K` |
| 竖版海报 | `4:5 + 2K` |
| 手机竖屏 | `9:16 + 2K` |
| 头像或方图 | `1:1 + 2K` |
| 最终精修图 | 对应画幅 + `4K` |

### 步骤五：添加参考图并指定角色

提示词助手里的参考图区域支持拖入、粘贴或点击上传，最多 6 张。

每张参考图都可以指定一个角色：

| 参考角色 | 用法 |
| --- | --- |
| 身份参考 | 锁定人物、角色或产品身份，避免被其它图混合替换 |
| 风格参考 | 只提取画风、质感、调色和整体视觉语言 |
| 构图参考 | 参考镜头角度、版式、空间关系和构图节奏 |
| 场景参考 | 参考地点、环境、背景叙事和空间氛围 |
| 产品参考 | 参考产品造型、结构、品牌文字和可识别特征 |
| 文字参考 | 参考画面文字、排版、标题风格或字形要求 |
| 材质参考 | 参考材质、表面处理、纹理和细节密度 |

默认情况下，第一张图会作为“身份参考”，后续图片会作为“风格参考”。如果你的意图不同，记得手动改一下每张图的角色。

### 步骤六：使用身份锁定

如果任务涉及人物、角色或产品一致性，建议打开“身份锁定”。

身份锁定会要求模型在英文 Prompt 中强调：

- 不改变年龄。
- 不改变脸型和五官比例。
- 不自动美化。
- 不混合其它参考图的身份特征。
- 保留产品或角色的核心可识别特征。

### 步骤七：填写规格补充

“规格补充”适合写不能遗漏的限制条件，例如：

```text
画面文字必须是“风起长安”。
不要出现现代建筑。
保持角色脸型、年龄和眼距不变。
品牌 Logo 必须清晰，不要改字。
```

### 步骤八：生成并检查结果

点击“生成提示词”后，右侧会出现“最终英文提示词”。复制前建议先看“中文核对”：

- 中文核对是否准确表达你的需求。
- 核对清单里主体、风格、镜头、画幅、分辨率是否都覆盖。
- “可能需要确认”里有没有需要补充的歧义。
- “负面约束”是否包含你不希望发生的变化。

确认无误后，点击复制按钮，把最终英文提示词粘贴到 Nano Banana Pro 中使用。

### 示例一：纯文字生成海报

推荐设置：

```text
模式：文生图
画幅比例：4:5
分辨率：2K
身份锁定：关闭
```

想法：

```text
生成一张国风武侠电影海报，主角站在雨夜古街中央，黑色斗篷被风吹起，远处有灯笼、雾气和湿润石板路，整体有电影级光影和高对比构图。
```

规格补充：

```text
不要现代建筑，不要卡通风，不要低清晰度。画面不需要文字。
```

### 示例二：用参考图保持人物身份

推荐设置：

```text
模式：图文生成
画幅比例：16:9
分辨率：2K
身份锁定：开启
```

参考图角色：

- 图片 1：身份参考
- 图片 2：风格参考
- 图片 3：构图参考

想法：

```text
根据图片 1 的人物身份生成一张电影剧照，使用图片 2 的冷色调和胶片质感，构图参考图片 3，让角色站在城市天台边缘，背景是雨后的霓虹夜景。
```

### 示例三：编辑图片

推荐设置：

```text
模式：编辑图片
画幅比例：1:1
分辨率：2K
身份锁定：开启
```

想法：

```text
保留原图人物的脸型、年龄、五官比例和发型，把服装换成黑色未来机能风外套，背景改成干净的摄影棚灰色背景，光线更柔和。
```

规格补充：

```text
不要改变人物身份，不要磨皮过度，不要增加多余配饰。
```

## 图片反推教程

1. 打开包含参考图的网页，或在侧边栏上传本地图片。
2. 在“图片反推工作区”选择本次要使用的模板，例如“风格提取”或“人物参考表”。
3. 右键网页图片选择“反推提示词”，或直接导入图片开始分析。
4. 首次使用时确认图片或图片 URL 可发送到配置的 API 服务。
5. 在 Prompt/JSON 双视图中查看结果。
6. 用自然语言继续修改，例如换主体、改色调、保留原图风格。
7. 复制 Prompt、复制 JSON，或保存到历史记录中复用。

## 适合场景

- AI 绘画参考图拆解。
- Midjourney、Stable Diffusion、GPT Image、Nano Banana Pro 等模型的提示词整理。
- 角色设定、道具设定、场景设定和美术外包 brief。
- 多张参考图的风格归纳和混搭创作。
- 团队沉淀固定格式的 Prompt 模板。
- 把中文创意需求改写成专业英文出图 Prompt。

## 对外短版介绍

AI Prompt Reverse Engineer 是一款面向 AI 图像创作者的浏览器侧边栏插件。它可以从网页图片、本地图片或多张参考图中反推出中文 Prompt 和结构化 JSON，也可以通过 Nano Banana Pro 提示词助手把中文想法、参考图和限制条件整理成可直接复制的英文提示词，并提供中文核对、模板库、混搭分析、自然语言修改、历史记录和 OpenAI 兼容网关配置，帮助用户把零散参考图快速整理成可复制、可编辑、可复用的生成提示词。
